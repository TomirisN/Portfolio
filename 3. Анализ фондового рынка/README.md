# Анализ фондового рынка

## Данные
Экспорт котировок Газпром АО, рынок Мосбиржа
https://www.finam.ru/quote/moex/gazp/export/

## Выполненные задачи
Из импортированных файлов  с информацией о динамике цен акций на фондовом рынке был собран датафрейм, осуществлена предобработка, добавлены метрики и проведен исследовательский анализ цен за последние 3 года. В исследовании использовались скользящие средние, экспотенциальные скользящие средние и RSI индикатор.

## Навыки и инструменты
Python, pandas, matplotlib, seaborn, визуализация данных, скользящие средние, RSI индикатор.
